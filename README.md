# AI Resume Analyzer

Live Demo:
https://ai-resume-analyzer-hktp.onrender.com

A Flask-based web application that analyzes resumes against job descriptions using NLP techniques.
## Features
- Resume PDF parsing
- Skill extraction
- Job match scoring using TF-IDF
- Missing skills detection
- Deployed web application

## Tech Stack
Python, Flask, scikit-learn, PyPDF2, Gunicorn, Render
